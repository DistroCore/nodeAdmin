# Ant Design Pro 前端重构设计

**日期**: 2026-03-19
**版本**: 1.0
**状态**: 草稿

---

## 1. 概述

### 1.1 目标

将 nodeAdmin 前端从基础 Ant Design 升级为 Ant Design Pro 风格，提供更专业的后台管理系统 UI。

### 1.2 技术栈

- React 18 + TypeScript
- Ant Design 5.x
- @ant-design/pro-components (ProLayout, ProTable, ProForm, ProList)
- React Router 6
- Axios (已有)

---

## 2. 组件映射

| 当前组件 | 替换为 | 说明 |
|---------|--------|------|
| Layout.tsx | ProLayout | 高级布局，含菜单、页头 |
| Table (antd) | ProTable | 带搜索、筛选、分页的高级表格 |
| Form (antd) | ProForm | 专业表单，支持验证 |
| 基础页面 | ProCard | 卡片式页面 |

---

## 4. 文件结构

```
frontend/src/
├── layouts/
│   └── ProLayout.tsx          # ProLayout 布局组件
├── pages/
│   ├── Dashboard/
│   │   └── DashboardPage.tsx  # 仪表盘
│   ├── UserManagement/
│   │   ├── UserListPage.tsx   # 用户列表 (ProTable)
│   │   ├── UserCreatePage.tsx # 创建用户 (ProForm)
│   │   └── UserEditPage.tsx   # 编辑用户 (ProForm)
│   └── RoleManagement/
│       ├── RoleListPage.tsx   # 角色列表 (ProTable)
│       ├── RoleCreatePage.tsx # 创建角色 (ProForm)
│       └── RoleEditPage.tsx   # 编辑角色 (ProForm)
└── components/
    └── RightContent/          # 右上角用户信息
        └── AvatarDropdown.tsx
```

## 5. 实施计划

### Phase 1: 依赖安装和基础布局
1. 安装 @ant-design/pro-components
2. 创建 ProLayout 布局组件
3. 修改 App.tsx 和 Router
4. 添加路由配置

### Phase 2: Dashboard 重构
1. 使用 ProCard 实现统计卡片
2. 使用 ProList 实现最近用户列表

### Phase 3: 用户管理重构
1. UserListPage → ProTable
2. UserCreatePage → ProForm
3. UserEditPage → ProForm

### Phase 4: 角色管理重构
1. RoleListPage → ProTable
2. RoleCreatePage → ProForm
3. RoleEditPage → ProForm
